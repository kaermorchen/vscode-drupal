# Тестирование расширения VSCode Drupal

## Структура тестов

- `unit/` - Модульные тесты для отдельных компонентов
- `integration/` - Интеграционные тесты
- `e2e/` - End-to-end тесты

## Запуск тестов

### Модульные тесты

```bash
npm run test:unit
```

### Все тесты

```bash
npm test
```

## Написание тестов

Тесты пишутся с использованием Mocha и Chai assertion library.
Файлы тестов должны иметь расширение `.test.ts`.
